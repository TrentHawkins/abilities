# Abilities

Display possible score palettes for a particular RPG setup.

Disclairmer:
The library `simple-term-menu` is used ([https://pypi.org/project/simple-term-menu/](https://pypi.org/project/simple-term-menu/)) but is included in the repository because it has been modified.

## Usage

Check viable scores for a character creation based on character hyperparameters.
* Select game from menu.

### Dungeons and Dragons

Before checking viable ability score palettes some hyperparameters need fixing:
* Select scoring tier. Beyond the standard are natural extensions/shrinkages of the standard.
* Select race and optionally subrace, to determine any bonuses that apply to your ability score palettes. You should be warned that some of the maximal sum abilitiy score palettes assume that you apply race bonuses to "expensive" score increments.
* Select extra ability points to accound for if loading a character of higher than 1 level. It measures in extra ability points instead of level, to account for custom character upgrading like getting a feature with only +1 or +0 ability points instead of the standard +2.

### Cyberpunk 2077

In this game 1 ability point is awarded per level:
* Select character level(s) to get viable ability score palettes and plan ahead your character development.
